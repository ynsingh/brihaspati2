Department and School/Center Registeration

Please fill all Mandatory(*) field.

Guideline For Department Registration:

Department Code- Write Department code (for example CSE-101).

Department Name- Write Department Name (ex:- computer Science and Engineering).

Department Nick Name- Write Department Nick Name (ex:- CSE).

Floors- Write how many floors in your department (Write Integer only).

Guideline For Map Department:

Deparment will be auto-mapped (check box disabled) when new Deparment is created by current institute.

For other institutes click on check box to select Department and click submit to map selected Department in institute.

Guideline For School/Center Registration:

School/Center Code- Write School/Center code (for example KV-201).

School/Center Name- Write School/center Name (ex:- Kendriya Vidyalaya).

School/Center Nick Name- Write Schol/Name Nick Name (ex:- CSE).

School/Center Description- Write short description about your School/Center.

Guideline For Map School/Center:

Select School/Center to be mapped.

Departments will be listed corresponding to selected school/center.

Choose Departments to be mapped with selected School/Center.

After Selecting both School/Center and Departments, click on submit to proceed mapping.

Choose No Department if School/Center dosen't contains any Departments and click submit to proceed mapping without Department.

Guideline For deleting Department/School:

Only Admin has privileges to delete Department/School.

Go in department/School list and click delete link to delete the Department/School.

After that click on remove link to confirm deletion of department/School.

Only that Department/School can be deleted which is used in single institute.
