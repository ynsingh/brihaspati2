Format for Marks List

The format for the marks list file should be as shown in the example below. The fields in any line should be separate by a comma (,). The first line should contain legends telling what the field is. The first column should be students rollno.

Roll.No, Name, Quiz-1, Quiz-2, Quiz-3, MS-1, MS-2, EndSem

Y3295, XYZ, 22, 25, 26, 0,0,45


NOTE :- Try to specify only those quizes and exams that are already been conducted and not for the ones to be conducted in future. If a student has not appeared for a particular exam, the instructor should put a space character in place of marks while making the file for uploading marks. Also the extension of the file should be ".txt". The format of the file should be as shown in the example below.

Roll.No, Name, Quiz-1, Quiz-2, Quiz-3, MS-1

Y3295, XYZ, 22,, 26, 30


The other student entries will also be similar to what is shown in line no.2. The legend line is immaterial for us, so you can use whatever characters you want except comma (,). We search the file using on first column entry hence what you put in other columns is also immaterial and it is shown as it is to the student.

To Convert Marks List in EXCEL to CSV format (Comma Seperated Version Format)

Follow the steps given below to convert the marks list created in Microsoft Office Excel Workbook to CSV format:-

1) Create the list in EXCEL in the format specified above.
2) Remember to include heading for example "Roll.No, Name, Quiz-1, Quiz-2, Quiz-3, MS-1" in the sheet.
3) To save the file go to 'File' menu and select 'Save As' option.
4) In the 'Save As Dialogue box', select the 'Save As type' to 'CSV (Comma delimited)', specify the filename and save the worksheet.
5) The file will be saved in the CSV format. Use this file for uploading the marks of the students of your course.
6) Upload : To upload marks file.
7) Attach File : To Attach multiple marks file add more option.
8) Remove File : Remove the option for adding more file.
