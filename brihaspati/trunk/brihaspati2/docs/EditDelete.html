Update and Delete Assignment


  Update Assignment

  1. Topic Name - The Instructor can Update Assignment Topic Name.
2. Due Date - The Instructor can change Due Date.
3. Instruction - The Instructor can change Instructions for their Assignments.
4. Max Marks - In this section, the Instructor change maximum Grade for their assignments.
5. Attach file - The instructor can change assignment's file which has to be only with the .txt , .html , .pdf extension.

Note : Availability option is optional, that means the instructor can check it or not too.

 

  Delete Assignment

  • The Instructor can Delete any Assingnment.
