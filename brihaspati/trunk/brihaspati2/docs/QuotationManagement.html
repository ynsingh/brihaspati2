Quotation Management

This module permits the Admin to manage the addition, updation and deletion of quotations. To add a quotation go to "Add Quotation" page. Write the quotation in the textbox and click the "Add" button. The quotations will be selected randomly and will be displayed on the Login page.

    Add Quotation

    To add a quotation go to "Add Quotation" page. Write the quotation in the textbox and click the "Add" button. The added quotation will be displayed in a list below the textbox. Each quotation will have the option of updation and deletion.

    Updating Quotation

    To update a quotation, click on the "Edit" button corresponding to the quotation that needs to be updated. Then a textbox will appear containing that quotation. The quotation can then be modified and changes will be saved by clicking on the "Update" button.

    Deleting Quotation

    To delete a quotation, click on the "Delete" button corresponding to the quotation that needs to be deleted. A popup window will then be displayed , confirming the deletion of the quotation. When "Ok" will be clicked then the quotation will be deleted from the list.
