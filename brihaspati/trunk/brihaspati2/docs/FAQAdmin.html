This page contain the list of all Category by clicking the category you get the list of all FAQ's. and additional links for Adding ,editing and deleting the FAQ.

In this page there are three links.

    1. Add Category

   Under this link add Category for the FAQ's and this link also show the list of category which are added previously.

  2. Add FAQ

   In this link by selcting any category add FAQ under that category.

  3. Delete Category

   This link show the list of all Category. by this link admin can delete the category.

  4. Edit Category

   This link show the list of all Category. by this link admin can Edit any category by changing its name.
