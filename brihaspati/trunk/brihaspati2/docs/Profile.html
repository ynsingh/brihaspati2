    1. Phone Number

   To enter a Basic or Dot number you have to fill country code (Restricted to 4 digit), region code (Restricted to 5 digit) and your phone number (Restricted to 10 digit).
To enter a mobile number you may left blank to region code.
There are three catagories of Phone number (Public, Protected, Private).
         Public : Any one can see these numbers.
         Protected : Only same institute member can see these numbers.
         Private : Only the person who is entering that no can access thease numbers.
