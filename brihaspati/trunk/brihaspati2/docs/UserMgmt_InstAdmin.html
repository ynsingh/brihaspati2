User Management

When we click on "User Management" following page is opened and some links shows such as:-

    Add User

    With this sub module, Admin can add user in respective course with roles. After filling all the data, press submit button and get the message on top of the screen "Registration successful in the specified course and Mail sent successfully". Finally you get the registration mail which contains server URL, user name and password.

    Student Course List

    Admin can Search a Student on the basis of his/her User Name, First Name, Last Name, E-mail & Roll No. The list of student are displayed with corresponding available actions (Course List). In course list link, get the list ofcourse which are opted by the corresponding student. From here institute admin can add or remove this student in any courses.

    Instructor Course List

    Institute Admin can Search a instructor on the basis of his/her User Name, First Name, Last Name & E-mail. The list of instructor are displayed with corresponding available actions (Update and Course List). In the update link institute admin can update the instructor first name , last name and email. In course list link, get the list of course which are opted by the corresponding instructor. From here institute admin can add or remove this instructor in any courses.

    Delete Instructor

    Admin remove instructor from any course at any time except primary instructor, for this get the list of instructor by selecting particular course. For removing particular instructor select that instructor from the list. When we select "All" option then list of Instructor along Course ID, User Name & Last Name are Shown with the "Remove" button. We can delete the whole instructor by this Remove button.

    View / Delete Student

    Admin can view and remove any student from any course. For this first get the list of student by selecting the course. For removing particular student select that student from the list. We can also choose "all" option for it. And if we select the option "All" the list of all students removed.

    Add Multiple Users

    Institute admin can upload multiple users at a time. The file should have ".txt" extension & proper format. The files contain username, password, first name, family name, email address and roll no all separated by semi-colons. Note that there is no semicolon after the rollno.The format is given below-

    email;password;firstname;familyname;programcode;rollno

    Each field in the entry should consist of alphanumeric characters. Email field is exception to this. '@' and '_' are allowed for use in this field. The username is the only mandatory fields without which the new entry into the system will not be registered. Similarly, an entry containing an illegal character will also not be registered into the system. Each entry must be in one line and must contain the five fields separated by four delimiters (;) or one field with five delimiters (;). To do so you have to fill course, role & attach file before clicking the upload button.

    Attendance Report

    This interface provides the last 30 days login time of all instructors to institute admin.
