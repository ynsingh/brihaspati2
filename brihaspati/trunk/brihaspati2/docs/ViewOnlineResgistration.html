View Online Registration

Click on (Online Registration) button. View Online Registration page will be open.

    Accept Button

    Select user or course and click on the accept button the user or course become active and selected user or course deleted from the page. Mail will be reached to him according his Email-ID.

    Reject Button

    Select the user or course and click on the reject button the selected user or course will be deleted from the page.
     

    Close Window
