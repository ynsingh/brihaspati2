Registered Courses

When the user insert correct user Id & password then page is opened where the following options are viewed:-

Course Id, Course Name, Status, Instructor Name, and Instructors E-mail. Anybody can send E-mails from here. Because the Instructors E-mail Id is given with them. We can also view the first, previous, next & last page of it because pagination feature is available in Brihaspati. We can also select value for pagination to show the required number of courses from 10 to 50. We can also search courses by their Id, if we give the correct Course Id only those Ids are viewed which are related to the given course Id.
