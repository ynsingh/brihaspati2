Guideline For Institute Admin Registration:

Institute Name- Write Institue name for example IITK or Indian Institute Of Technology Kanpur, ... etc.

Institute Address- Write Institue address.

City- Write city name where your institue belong.

Pincode- Write your six digit area pincode.

State- Write name of state.

Landline No- make sure all the fields of land line no filled correctly.

  * Country Code- Write country code for example :- 91,...etc

  *Note :- Don't add + sign before country code.

  * Region Code - Write Region Code for example :- 0512,0522, ...etc

  * Number - Write Phone Number for example :- 2597070,2598759, ...etc

Institute Domain- Write domain name of your website for example iitk.ac.in, iitg.ac.in, ....etc

Type Of Institute- Select any type Of institute from given option for example if your intstitute type is "govt." then select "govt." option.

Institute Website- Write your institute website name for example:- www.iitk.ac.in, ...etc

Admin's First Name- Write first name of institute admin.

Admin's Last Name- Write Last name of institute admin.

Admin's Designation- Write Admin's Designation.

Admin's Email- Write valid email address so that email reach to you and use as login name.

Admin's Password- Write secure password so that only institue admin used for login and access of content of brihaspati.
