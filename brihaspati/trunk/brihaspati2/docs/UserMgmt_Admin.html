1. View Admin's List shows the list of all Admin including (Primary Admin and Secondry Admin ) in an Institute. Green Color(First Row) shows Primary Admin of Institute. This Primary Admin can Register another Admin (Role as a Secondary Admin).

2. Primary Admin can Enable Permission(Change Role from Secondary Admin to Primary Admin). It Shows in Pink Color Row. Other reamaining Rows show Secondary Admin.

3. Primary Admin can also Disable Permission(Change Role from Primary Admin to Secondary Admin).

4. Secondary Admin can only View List of Admin.

5. Primary Admin can Remove All Admin(Primary and Secondary) Except Primary Admin from Institute.
