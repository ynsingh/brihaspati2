Courese Wiki
Wiki is a module which is used for creation of wiki based course content and allows all registerd users
for creating your own page or add/update/modify page whose content is allredy exist in course-wiki
see only pages of that course in whch you are registered.
For Instructor-
Instructor can create, update and delete any content under any topic.
Wiki Home -
When user click Wiki Home button then it open wiki default page which named same as course name. 
If user can create new page then he create topic name Eg.[[Demo]] and save content after that he click Demo 
then open a blank page, he will click edit button and write content in textarea and click save button.
Index -
After clicking index button user can see all topics, Edit content and check logs by clicking View Log details. 
Search -
This is used to search any key word in Index.
Control Wiki - 
In this page instructor can view all logs and delete any particular topic. 
If instructor wants that no one can edit his content, first he has to select topic name in droup down list
and click log button it will show Disable/Enable button he can disable edit by clicking disable button  
disabled content can not be edit by instructor also if he want edit again he has to click edit enable button 
 Student face-
Student can create and update topic.
