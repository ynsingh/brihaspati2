How to use Spreadsheet

Instrurctor can upload marks due to following steps.

Step1: The first line should contain legends telling what the field is. The first column should be rollno of student on Brihaspati system.The format for the marks list file should be as shown in the example below.

Roll No, Name, Quiz-1, Quiz-2, Quiz-3, MS-1, MS-2, EndSem

Y123, XYZ, 22, 25, 26, 0,0,45

NOTE :- Try to specify only those quizes and exams that are already been conducted and not for the ones to be conducted in future. If a student has not appeared for a particular exam, the instructor should have to leave a blank cell in place of marks .

Step2: You can put SUM , AVERAGE , MIN & MAX formula in the cell as given below.

=sum(a1:b1) , =average(a1:b1) , =max(a1:b1) & =min(a1:b1) (formula is not case sensitive) and formula is saved after clicking on save button.

Step3: Before saving or updating spreadsheet you have to click or select a blank cell.
