Tracking Reports

In this page there are three links.

    1. User Reports

   Under this link institute admin can track report of any user by selecting the user name. it provide the all detail of particular user (like username,userrole,joining date and coursename etc ).

  2. System Reports

   Institute Admin can track reports by three ways(Show All Users,Show All Institute and Show All Courses). under this institute admin track reports by selecting the date. admin can get the monthly report of any user (like total access time and total no of login) or any course(total no of users access and total access time).

  3. Most Active User and Most Active Course

   This link provide the information of Most Active user and Most Active Course.
