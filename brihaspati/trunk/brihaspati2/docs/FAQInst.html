This page contain the list of all FAQ's by clicking the question you get the answer of that question and also give the answer of that question by clicking the link Add Answer and additional links for editing and deleting the FAQ in case the user is an instructor/student in the particular course.

In this page there are three links.

    1. Add FAQ

   Under this link Instructor can add FAQ and give the expiry date for rating the FAQ which is rated by the student.

  2.. Favorite FAQ

   This link show the list of favorite FAQ's which added by the instructor. Instructor can also set FAQ as a unfavorite by selecting any FAQ.

  3.. Select Favorite

   This link show the list of all unfavorite FAQ's. by this link instructor can set FAQ as a Favorite.
