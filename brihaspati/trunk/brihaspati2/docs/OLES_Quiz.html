Generate Exam

This page provides an interface by which the instructor can Generate Quiz/Exam of a course.
This Page contain four links.

1.Quiz Setup

2.Create Quiz

3.Quiz Detail

4.Preview

1.  Quiz setup
Instructor create the quiz setup by entering quiz name, maximum marks, maximum number of questions and maximum time to attempt the quiz. QuizID is autogenerated and can not be changes. quiz name should be unique.

2.  Create Quiz
After quiz setup... quizs are populated in the list box...
After selecting a quiz, control moves to quiz mode selection...quiz creation modes are two -->

  • Random
  • One by One

Random
In the random quiz creation,instructor is asked to enter topic, Type of question, Difficultylevel, mark per question and number of question. On the basis of the given criterion, question are selected from the question repository.
On the click of "Save Question & add more", same page displays again and ask to enter values.
On the click of "Save Question & Exit", control goes to generate exam page.

One by One
In One by One quiz creation mode,instructor is asked to enter topic, Type of question & Difficulty level. Upon clicking "search" button, on the basis of the specified criterion question are displayed from the question repository.after selecting the question & clicking add, a new window open with given setting and here instructor can write marks for the selected question.Finally clicking Add, question is added in the quiz

3.  Quiz Detail
This tab displays list of created quiz. Every created quiz has 3 options


  • Add more Question
  • Delete
  • Update
    • Update Quiz Setting
    • Update Question Setting
Add More Question
    More questions can be added in the already created quiz
Delete:
    Quiz is deleted and all its saved questions file are also deleted.
Update:
    this tab has two sub menu.
1.Update Quiz Setting
2.Update Question Setting
1.Update Quiz Setting

    Quiz settings entered during quiz setup are updated through this option.


2.Update Quiz Setting

    Quiz Question's settings entered during quiz creation are updated through this option.

4.  Preview
This tab is available if any quiz is randomly created..two options are available here : 1.Accept 2.Reject
if instructor is satisfied with the choosen setting then accept button will save the setting and reject button click will remove the settings..
