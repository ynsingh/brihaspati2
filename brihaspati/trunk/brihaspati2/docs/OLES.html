Online Examination System

This page provides an interface by which the instructor can create Question Bank repositories in course.
The given link Create new Question bank: This Page contain three links.

1.Add Multiple Questions

Instructor upload the multiple questions by selecting Question type and difficulty level. Add multiple questions by uploding the file(.txt) which contains the multiple question.

The format for uploading the multiple question is

Questionid;Question;optionA;optionB;optionC;optionD;Answer(write option only);Questiondescription

NOTE (This note is only for the Answer):
1. OptionA means "A"
2. OptionB means "B"
3. OptionC means "C"
4. OptionD means "D"
Example (Multiple choice question type):

1;what is brihaspati?;lms;cms;bms;dms;A;opensource learning management system

NOTE:

Questionid =1
Question =what is brihaspati?
OptionA =lms
OptionB =cms
OptionC =bms
OptionD =dms
Answer =A
Questiondescription =opensource learning management system

Questionid;Question;Answer;Questiondescription

Example (True/False question type, Short answer type and Long answer type):

1;what is brihaspati?;lms;opensource learning management system
NOTE:
Questionid =1
Question =what is brihaspati?
Answer =lms
Questiondescription =opensource learning management system

2.Upload Image type question

Instructor upload image type question by selecting the question type and difficulty level and attach the image of the question.


3.Add one by one Question

If instructor wants to add the question according to the choice then instructor add question one bye one with the question type and difficulty level.


View Question:

    View the questions according to the topic by selecting Question type nad difficulty level.


Edit Question:

    Instructor can edit the uploaded questions by selecting the topic with the Question type and difficulty level.


Delete Question:

    Instructor can delete the questions from the question bank accroding to topic.


NOTE:
Question type :
1.Multiple choice question type
2.True/False question type
3.Match question type
4.Short answer type
5.Long answer type
Difficulty level:
1.Easy
2.Medium
3.Hard
