List Management

There are two links in this page.

    View All Course

    When we click on, "View All Course" the whole list of the users with the Course ID, Course name, Department, Description, Creation Date and Status are shown. With all these User's list the "Edit" option is also visible with each description and from here we can edit any field. We can also search them by their Course ID, Course name & Department. And we can also search Course list by giving the Match string.

    View All Users

    When we click on, "View All User" whole list of the users with the User Name, Name, Email ID, Available Actions, are shown. When we search it by any specific User Name then the following window is open with Update, Change Password, Remove, Usage Details options.
