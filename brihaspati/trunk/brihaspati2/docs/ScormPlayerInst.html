Upload Scorm Compliant Package

Upload Scorm compliant package: Instructor can upload scorm compliant package due to following steps.

Step1: First we check the scorm package is zip formate or not.

Step2: After checking zip formate we are checking imsmanifest.xml file exit in zip file.

Step3: Then we check Imsmanifest.xml file is scorm complaint or not.

Step4: After checking all previous step we can upload scorm package in course area.

Instructor can view scorm package file and some available action is given below:

Delete: Delete the upload scorm content package using delete link. Show Content Instructor can also view scorm compliant package.

Launch: The Instructor can launch scorm package for his/her student. so the Instructor can click Launch link then content is launch,this content can be view by the students.

Unlaunch: Instructor can also unlaunch the scorm content i.e the scorm content package is disable for his/her course content area.whenever instructor can unlaunch the content, student not able view the content.
